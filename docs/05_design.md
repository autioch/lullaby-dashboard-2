# Dashboard Specification

## Layout

### Header

Mission name.

### Progress Area

Large visual progress indicator.

Visible from distance.

### Status Message

Generated from progress.

### Remaining Objectives

Only remaining objectives are displayed by default.

Completed objectives should:

- Move to bottom
- Collapse
- Fade visually

### Timing

- Mission duration
- Optional deadline countdown

### Records

- Best completion time

## Theme System

Themes affect presentation only.

Themes must never alter mission logic.

### Initial Themes

#### Space

- Mission Control
- Crew Boarding
- Launch Ready
- Orbit Achieved

#### Minecraft Adventure

- Village Preparation
- Supplies Ready
- Entering Cave
- Expedition Complete

#### Pirate Voyage

- Crew Boarding
- Raise the Sails
- Leaving Port
- Treasure Expedition Complete
