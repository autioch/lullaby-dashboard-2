# Lullaby Dashboard 2

Small Astro-based dashboard app named Lullaby dashboard.

What it does

- loads a set of predefined routines from Firebase
- shows a selectable list of routines
- displays: a checklist of tasks, an embedded YouTube video, a live clock

## Development

Dev server: `npm run dev`
In the browser: http://localhost:4322/
Create a `.env` file in the project root based on the `.env.example` file.

## Tech stack

- Astro
- React
- Zustand
- Firebase
